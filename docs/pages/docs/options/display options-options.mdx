# Display Options Options

## --hidePageHeader

<Callout emoji="ðŸ’¡">Do not print page header.</Callout>

> Accepts a boolean value. Defaults to `false`.

```json filename="typedoc.json"
{
  "hidePageHeader": false
}
```

## --parametersFormat

<Callout emoji="ðŸ’¡">
  Specify the render style of parameter and type parameter groups.
</Callout>

> Accepts one of `"list"` | `"table"` | `"htmlTable"`. Defaults to `"list"`.

