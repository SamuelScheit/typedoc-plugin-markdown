# Quick Start

## Installation

Install Docusaurus in the root of your project and install the plugin dependencies in the same location as the Docusaurus website directory.

```sh npm2yarn
npm install typedoc typedoc-plugin-markdown@next docusaurus-plugin-typedoc@next --save-dev
```

## Usage

<Steps>

### Add the plugin to `docusaurus.config.js`

Add the plugin to `docusaurus.config.js` and specify the required options.

The following options can be passed to the config:

- [typedoc](https://typedoc.org/options) options.
- [typedoc-plugin-markdown ](/options) options.
- [Plugin](/integrations/docusaurus/options) options.

TypeDoc options can also be declared:

- Using a `typedoc.json` file.
- Under the `typedocOptions` key in `tsconfig.json`.

Please note that options declared in this manner will overwrite options declared in `docusaurus.config.js`.

```js filename="docusaurus.config.js"
module.exports = {
  plugins: [
    [
      'docusaurus-plugin-typedoc',

      // Plugin / TypeDoc options
      {
        entryPoints: ['../src/index.ts'],
        tsconfig: '../tsconfig.json',
      },
    ],
  ],
};
```

### Run Docusaurus

You can generate the docs by either:

- Running Docusaurus by using the `start` or `build` [cli commands](https://docusaurus.io/docs/cli#docusaurus-cli-commands).
- Building the docs independently by running the custom cli command `docusaurus generate-typedoc` and then running Docusaurus at another time.

### View the docs

Once built the docs will be available at `/docs/api` (or equivalent output directory).

e.g. `http://localhost:3000/docs/api`

</Steps>
