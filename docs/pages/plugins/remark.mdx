# typedoc-plugin-remark

<PackageDescription pjson={pjson} />

## Overview

This plugin is useful if additional markdown processing is needed after the markdown files are generated by TypeDoc using the popular [remark](https://github.com/remarkjs/remark#readme) plugin system.

## Features

- Pass in a list of remark plugins to the TypeDoc config.
- Write your own custom plugin.
