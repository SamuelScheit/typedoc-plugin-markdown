# Options

These are the additional options that can be added to the TypeDoc configuration.

## remarkPlugins

<Callout>An array of remark plugin names.</Callout>

> Accepts an Array.


Each required plugin should be individually installed.

Please note options can be passed either as an array of strings or an array of string / options.


```json filename="typedoc.json"
{
  "remarkPlugins": [
    "unified-prettier",
    "remark-github",
    [
      "remark-toc",
      {
        "maxDepth": 3
      }
    ]
  ]
}

```
