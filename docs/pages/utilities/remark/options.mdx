# Options

These are the additional options that can be added to the TypeDoc configuration.

## remarkPlugins

<Callout>An array of remark plugins.</Callout>

> Accepts an Array.


```json filename="typedoc.json"
{
  "remarkPlugins": [
    "remark-github",
    "unified-prettier"
  ]
}

```
